== Description ==

Secure your website with the most comprehensive WordPress security plugin. Firewall, malware scan, registration blocking, login security & more.

The plugin tracks all sign in attempts and sign up attempts (new user and new blog) and block it if the user has reached limits.

The plugin is translated into **russian**, **german** and **english**.

LOGIN AND REGISTRATION ATTEMPTS LIMIT works fine in **WordPress Multisite** too and supports global configuration (networkwide active) and per blog (if it is activated in the blog).
